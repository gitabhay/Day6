Ques. 4 : Explain different types of CSS.

There are three ways to include our css.
  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS :-

In inline css we need to mention our styling within the opening tag only and we need to use style attribute.

Inline CSS

Internal CSS :-

Internal is having more priority than the external. Internal css should be written inside only in our head tag.
Style tag is need to be used in our Internal css. Whenever we are using class we need to call with . (dot) Example:- .box .example. Whenever we are using id we need to call with # Example:- #box #example

Try to use Inline styling when you want to apply one or two properties.

Class:- Class can be used multiple times in an element. Id :- Id can be only in our element.

Internal CSS

External CSS :-

When ever we are using external style sheet we should link with the link tag. Multiple class is applied but only one id is applied for css.

Ques. 5 : Create an example to explain the difference between margin and padding.

Margin :- The margin between the Browsers screen and the Element border.
  • Margin left
  • Margin Right
  • Margin top
  • Margin Bottom
Short Hand Properties :- Margin:5px 10px;   5px is going to be top and bottom   10px is going to be right and left

Margin:10px 5px 10px 5px;   Top Right Bottom Left

Padding :- The space between the Element border and the Content.
  • Padding left
  • Padding Right
  • Padding top
  • Padding Bottom
Short Hand Properties :- Padding:5px 10px; 5px is going to be top and bottom 10px is going to be right and left

Padding:10px 5px 10px 5px;   Top Right Bottom Left
Padding & Margin
